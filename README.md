# 🚀 AI-Powered Smart Job Portal (Backend)

This is a professional Django-based Job Portal Backend featuring **REST APIs** and **AI-driven automation**.

### ✅ Key Features (Accounts Completed)
- **Custom User Model**: Roles for Seeker and Employer.
- **Token Authentication**: Secure login using Django REST Framework Tokens.
- **Role-Based APIs**: Separate registration for Seekers and Employers.
- **Profile Management**: Secure API endpoint to get/update user profile.

### 🛠 Tech Stack
- **Language**: Python 3.x
- **Framework**: Django 5.x
- **API**: Django REST Framework (DRF)
- **Database**: SQLite (Development)
